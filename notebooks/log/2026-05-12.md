## 5. 02 train/val 切分 → time-based

之前用 `random_split` → val/train 在时间上交错 → soft leakage（评估偏乐观）。改成：

```python
train_set = Subset(dataset, range(n_train))           # 前 80% = 早期
```

## 9. Mean-variance baseline

[src/baselines/markowitz.py](../../src/baselines/markowitz.py) 已加入 FinCast mean-variance baseline：

- 从 return patch 的近端 horizon 估计 `mu` 和 `sigma^2`
- `mu` 默认取 FinCast mean 的前 `K=5` 步平均
- `sigma` 默认用 `(q90 - q10) / 2.563` 近似，再对前 K 步方差取平均
- 闭式仓位为 `p = mu / (risk_aversion * sigma^2)`，再做 long-only clip、`max_trade` 限制和 `round_step` 离散化
- 默认约束和 trader 对齐：`position in [0,1]`、`max_trade=0.25`、`round_step=0.01`
- 已补 [tests/test_markowitz_baseline.py](../../tests/test_markowitz_baseline.py)；当前环境没有 `pytest`，但直接调用测试函数通过
